Arithmetic Operations in C++

This is a simple C++ program that performs basic arithmetic operations (Addition, Subtraction, Multiplication, Division) on two numbers provided by the user.

🚀 Features

- Takes two numbers as input
- Performs:
  - Addition
  - Subtraction
  - Multiplication
  - Division

🛠️ Language Used

C++

▶️ How to Run

1. Compile the code:
   g++ .cpp
2. Run the program:
   ./a.exe

📌 Example

Input:
Enter two numbers: 10 5

Output:
Addition: 15
Subtraction: 5
Multiplication: 50
Division: 2

📚 Learning Purpose

This project was created while learning Data Structures and Algorithms (DSA) to understand basic input/output and operations in C++.
